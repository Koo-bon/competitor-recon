# Rise-to-Fame Deconstruction — *why did this blow up?*

A selling point tells you *what* a brand offers. This module reverse-engineers
*how it actually became popular* — the growth mechanism. This is what separates a
shallow competitor list from real strategic intelligence. Do this for the 2–3
breakout competitors (the ones the user is most inspired/intimidated by).

## The core question
"If I had to recreate this brand's rise, what were the specific moves — and which
of them were repeatable vs. luck?"

## The 6 growth levers to trace (find evidence for each; mark ✅/🟡/⚠️)

1. **Origin spark** — what was the first moment of traction? (a viral post, a
   celebrity carry, a collab, a limited drop). Date it if possible.
2. **Ignition channel** — where did it spread? (TikTok unboxing, Instagram
   Reels, K-pop idol sighting, resale hype on Xianyu/Bunjang, Reddit). Name the
   platform and the format.
3. **Scarcity / mechanic** — what structural device drove repeat demand? (blind-box
   randomness + secret rarity, timed drops, numbered editions, size ladders).
4. **Community / identity** — what made people *belong*? (collector culture,
   trading, display shelves, fandom). People stay for identity, not product.
5. **Collaboration / borrowed audience** — which partners lent credibility/reach?
   (FIFA, Casetify, artists, celebrities). Note who they borrowed from.
6. **Emotional core** — the one feeling the brand reliably delivers (healing,
   status, nostalgia, cuteness). This is the durable engine under the tactics.

## Build a timeline
Lay the levers on a simple dated timeline so the *sequence* is visible — order
often matters more than any single move:

```
2015 ── origin: niche designer toy
2019 ── ignition: celebrity carry on Instagram
2023 ── mechanic: blind-box + secret drives resale
2024 ── collab: global IP partnership → mainstream
2025 ── community: shelf-display + trading culture peaks
```

## Separate signal from luck
For each lever, tag it:
- 🔁 **Repeatable** — a move the user could deliberately engineer.
- 🎲 **Circumstantial** — luck/timing unlikely to recur.

The payoff line: *"The repeatable levers behind [brand]'s rise were X and Y; the
part you can't copy was Z."* That's the sentence the user actually needs.

## Visibility
Present each breakout competitor as a compact **rise card**:

```
### {Brand} — how it blew up
Emotional core: {one feeling}
Origin spark → Ignition → Mechanic → Collab → Community   (one line each)
Timeline: {dated arrow}
Repeatable for us: {🔁 levers}   |   Can't copy: {🎲 levers}
```
